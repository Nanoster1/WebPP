О нас

CSOL - платформа, позволяющая играть на любых платформах в нетребовательные веб-игры. Главной особенностью является то, что нет необходимости дополнительно устанавливать вспомогательные программы. Все происходит прямо на сайте. API сайта позволяет разарботчикам более тесно взаимодействовать пользователям, давая все необходимые инструменты для этого.
